About

I currently work as a software engineer at SALT, a mobile app for remembering, sharing, and discovering great restaurants and bars. It's honestly the best app out there to help you keep track of the thousands of restaurants you come across in big cities like New York. The team at SALT is awesome and I feel honored to be working with them.

I do true full-stack development. My areas of expertise include Node.js API design and implementation with Express, relational databases, and front-end iOS development with Swift. I also have a working knowledge of Angular 1 and Bootstrap, and I've dabbled in React but am yet to use it in a project.

Prior to SALT, I worked as a Java developer at ClearFactr, a web application for creating clearer and more robust financial models. ClearFactr was founded by my dad. He built the entire application himself (well, he's received some help from some world-class developers along the way) and continues to put his blood, sweat, and tears into it. Check it out if you're in the market for a tool to help you with financial modeling.

Location: New York, NY

Education: University of Rochester, B.S. Mechanical Engineering, 2014

Favorite tech: JavaScript, Node.js, Swift, SQL, AWS
